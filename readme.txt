INTRODUCTION
------------

The application uses the MVC pattern, take array and prints the following ASCII table.

REQUIREMENTS
------------

The application doesn't require any models.
The application doesn't need any form of modification (e.g. configuration).

CONFIGURATION
-------------

The application processes a mixed two-dimensional array of this order:

array(
    array(
        'House' => 'Baratheon',
        'Sigil' => 'A crowned stag',
        'Motto' => 'Ours is the Fury',
        ),
    array(
        'Leader' => 'Eddard Stark',
        'House' => 'Stark',
        'Motto' => 'Winter is Coming',
        'Sigil' => 'A grey direwolf'
        ),
    array(
        'House' => 'Lannister',
        'Leader' => 'Tywin Lannister',
        'Sigil' => 'A golden lion'
        ),
    array(
	      'Q' => 'Z'
    )
);
